vibe tools

Small standalone HTML tools

📝 Text
flow — Write like you're running out of time. charcount — Count characters and words in a text.
🖼️ Images
caption — Add readable text overlays to images. palette — Extract dominant colors from any image.
💪 Fitness
squatcount — Count squats with your webcam.
⏱️ Productivity
countdown — Count down to any moment, or run a Pomodoro timer.
📊 Visualizations
temperature — Explore how temperature scales relate.
🎮 Mini-games
echomaze — Find the exit by sound. wallflipper — Ride, flip, and survive a neon corridor at breakneck speed.
🤖 LLM tools (might require API key)
minority report — Ask up to three LLMs the same question, side by side. tokenopus — Count tokens using the Anthropic API (Opus 4.6 & 4.7). tokinem — Count tokens based on Eminem lyrics.
